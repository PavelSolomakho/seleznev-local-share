# SELEZNEV Local Share 2.0

Первый вход: admin / 1234
После запуска сразу смените пароль в меню «Пароль».

Функции:
- авторизация
- комнаты/папки
- история действий
- drag & drop
- превью фото
- рендер STL
- 3D просмотр STL в браузере
- чат
- QR-код
- PWA для телефона

Запуск:
1. Распакуйте архив
2. Запустите start.bat
3. Откройте http://127.0.0.1:8000

С телефона: http://IP-КОМПЬЮТЕРА:8000

Важно: 3D STL использует Three.js через интернет-CDN. Рендер STL картинкой работает локально.

## Пользователи

В меню «Пользователи» администратор может создавать пользователей, менять пароли и удалять пользователей.
